# Personal Portfolio — HTML / CSS / JavaScript

## Description
Development of a **modern and responsive personal portfolio website**, designed and coded entirely from scratch using **HTML**, **CSS**, and **JavaScript**.  
The project showcases academic background, technical skills, and personal projects, and is hosted on **Vercel** for public access.

The portfolio was created to serve as a professional online presence during the **third year of the Computer Science Bachelor’s degree** at the **University of Lorraine**, and as a platform for internship applications.

## Technical Overview
The website emphasizes both **performance** and **design consistency**, built with:
- **Semantic HTML5** structure for accessibility and SEO  
- **Modular CSS** using custom properties and responsive units  
- **JavaScript modules** managing interactive behaviors such as:
  - Persistent **sticky header** and active section highlighting  
  - **Dark/light theme switching** with color gradients adapting to mode  
  - **Smooth scrolling and responsive layout adjustments**  
  - Dynamic rendering of project cards and external links  

All assets (logos, icons, CV, and media) are organized in a dedicated `/resources` directory for easy maintenance and updates.

## Design Principles
The portfolio design focuses on:
- **Clarity and simplicity**, with a clean layout and clear hierarchy  
- **Visual coherence**, using a limited, high-contrast color palette  
- **Accessibility**, ensuring readability across devices and themes  
- **Scalability**, allowing new projects or sections to be added easily  

## Hosting & Deployment
The site is deployed via **Vercel**, offering continuous deployment and SSL by default.  
It is optimized for both desktop and mobile navigation, with fast load times and lightweight assets.

## Learning Objectives
This project reinforced skills in:
- **Front-end web development (HTML, CSS, JavaScript)**  
- **Responsive and accessible design**  
- **Code modularization and DOM manipulation**  
- **Version control and deployment workflows**  
- **Personal branding and UX design**
